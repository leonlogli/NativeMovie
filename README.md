# NativeMovie

![Demo](https://github.com/leonlogli/NativeMovie/blob/main/screenshots/demo.gif)

React native movie app using react-native cli

### Dependencies

```
yarn install
```

### Environnements

Rename .env.sample to .env

## Build android/ios

```
yarn android
yarn ios
```
